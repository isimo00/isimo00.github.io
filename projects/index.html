Under supervision

Some notable projects my code has contributed to (not including my BSc and MSc thesis):

  • soupy library for Stochastic Optimization under high-dimensional Uncertainty in Python, developed by the Scientific Computing and Uncertainty Quantification @ GeorgiaTech. Github repo (Python) 2024
  • Study on the union of hyperbolic graphs. Private repository, under Prof. Maria Serna’s supervision. 2023-24
  • biggr library, developed by BEEGroup @ CIMNE. Github repo (R) 2023
  • beemeteo library, developed by BEEGroup @ CIMNE. Github repo (Python) 2023
  • Wall modelling using a MLP neural network project for turbulent flow boundary layer modeling. Private repository, under Prof. Bernat Font’s supervision @ BSC-CSN. (Python) This was an extracurricular extension of my aerospace engineering bachelor’s thesis (for diploma obtention). 2021-22
  • Various \(\LaTeX\) workflows for automation and templates. Private repository, under Prof. Rafael Weyler and Prof. Josep Piñol’s supervision @ UPC, Material Science department. 2021
