Some notable projects my code has contributed to:

  • biggr library, developed by BEEGroup @ CIMNE. Github repo (R)
  • beemeteo library, developed by BEEGroup @ CIMNE. Github repo (Python)
  • Wall modelling using a MLP neural network project for turbulent flow boundary layer modeling. Private repository, under Prof. Bernat Font’s supervision @ BSC-CSN. (Python) This was an extracurricular extension of my aerospace engineering bachelor’s thesis (for diploma obtention).
  • Various $\LaTeX$ workflows for automation and templates. Private repository, under Prof. Rafael Weyler and Prof. Josep Piñol’s supervision @ UPC, Material Science department.
